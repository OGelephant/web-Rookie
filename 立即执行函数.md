### 立即执行函数
也就是只针对初始化功能的函数
执行完之后立即释放
也是js中提供的唯一一个能执行完立即销毁的函数。
但是依然有返回值  



形式 :
 （function (){}()）-->**w3c推荐**  
 
或者
（function(){}）();

函数声明
function test (){
  var a = 123
}

>1.只有表达式才能被执行，上面的只是声明。<br>
>2.能被执行的表达式函数名字就会被自动忽略<br>
所以即使例如以下这种形式也可以被执行，效果和立即执行函数相同。


```  
+function test () {
     console.log('a')
  } 
  ```
